A modern web-based FMCG product distribution platform designed for wholesalers and retailers to browse products, manage orders, and streamline product management through an integrated admin panel.
</p>

## 📌 Overview

Super Star Agencies is a digital FMCG distribution platform designed to help wholesalers showcase and manage their product catalog efficiently.

The platform provides a **customer-facing product browsing system** and an **admin management panel** that allows administrators to add, update, and manage product listings.

Core concepts demonstrated in this project:

• Product catalog management<br>
• E-commerce style product browsing<br>
• Category and brand filtering<br>
• Shopping cart system<br>
• WhatsApp-based checkout system<br>
• Admin panel for product management<br>

The system enables retailers to easily explore FMCG products and place orders directly through WhatsApp.

## 🧩 Features

### 🛍 Product Browsing
• Search products or brands<br>
• Category-based filtering<br>
• Product listing with pricing<br>
• Organized product catalog<br>

### 🛒 Shopping Cart
• Add products to cart<br>
• Quantity increase/decrease controls<br>
• Remove items from cart<br>
• Automatic order summary calculation<br>

### 📦 Checkout System
• Delivery information form<br>
• Customer name, phone number, and address input<br>
• WhatsApp checkout integration for placing orders<br>

### 🧑‍💼 Admin Panel
• Secure admin login system<br>
• Add new products<br>
• Update existing products<br>
• Delete products<br>
• Manage product categories<br>

### 📊 Product Management
• Category-based organization<br>
• Product pricing management<br>
• Image upload for product listings<br>

## 🛠 Tech Stack

• Frontend: HTML5, CSS3, JavaScript<br>
• Backend: Python (Flask) / Server-side API<br>

• Authentication: Admin login system<br>

## ⚙️ Setup Instructions

**1️⃣ Clone the Repository**

git clone https://github.com/yourusername/super-star-agencies.git

cd super-star-agencies


**2️⃣ Install Backend Dependencies**

pip install -r requirements.txt


**3️⃣ Run the Backend Server**

python app.py


**4️⃣ Open the Application**

Open your browser and navigate to:

http://localhost:5000
